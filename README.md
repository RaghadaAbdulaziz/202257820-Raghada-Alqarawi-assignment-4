# Personal Portfolio Web Application

## Overview
This project is a personal portfolio web application developed as part of Assignment 4. It combines previous assignments into one complete, interactive, and responsive website.

The website showcases personal information, projects, and multiple interactive features, while focusing on clean design and user experience.

---

## Features

- Responsive design (works on mobile and desktop)
- Dark/Light theme toggle (saved using localStorage)
- Project filtering by category
- Project search functionality
- Project sorting (A–Z, Z–A)
- Modal popup for project details
- Developer Advice API integration
- Contact form with validation and feedback
- Save visitor name using localStorage
- Session timer
- Scroll-to-top button
